# Trading System Simulator

A Python-based trading system simulator with GUI interface for analyzing trading strategies through Monte Carlo simulations.

## Features
- Multiple simulation runs
- Risk-reward analysis
- Win rate calculations
- Maximum drawdown analysis
- Interactive GUI interface
- Customizable parameters
- Visual results with charts and graphs

## Installation

1. Clone the repository:
   \`\`\`bash
   git clone [your-repo-url]
   cd trading_system-simulator
   \`\`\`

2. Create a virtual environment:
   \`\`\`bash
   python -m venv env
   source env/bin/activate  # On Windows: env\\Scripts\\activate
   \`\`\`

3. Install requirements:
   \`\`\`bash
   pip install -r requirements.txt
   \`\`\`

## Usage

Run the GUI application:
\`\`\`bash
python GUI_app.py
\`\`\`

## Requirements
- Python 3.x
- Required packages listed in requirements.txt
